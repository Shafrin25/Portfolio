## 👩‍💻 About This Project
This repository contains my **personal portfolio website**, designed to showcase my projects, technical skills, and development approach through a clean, structured, and interactive interface.

The portfolio emphasizes **modern UI design, smooth animations, responsive layout, and clear visual hierarchy** to present my work in a professional format.

- Responsive modern portfolio design
- Animated sections using **AOS (Animate on Scroll)**
- Interactive project preview videos
